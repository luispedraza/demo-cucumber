1. Crear un proyecto node

   `npm init -y`

2. Instalar Cucumber

   `npm i -D @cucumber/cucumber`

3. Instalar chai

   `npm i -D chai`

4. Instalar servidor http

   `npm i -D http-server`

5. Instalar selenium

   `npm i -D selenium-webdriver`

6. Instalar chromedriver

   `npm i -D chromedriver --global`

- Opcional: Crear repositorio

  `git init`
